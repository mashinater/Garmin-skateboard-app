# Garmin Skateboarding App

A lightweight skateboarding activity tracker for Garmin Instinct watches.

Features
- GPS speed tracking
- Session timer
- Distance tracking (planned)
- Push detection (future update)
- Trick detection (future update)

Goal:
Create a dedicated skateboarding activity for Garmin users.
